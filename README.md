# Finance Tracker 🪙

Minimalist engine to monitor cashflow.

## ⚡ Core Specs
* **Instant Logging:** Snap-fast entry of spends.
* **Smart Balancer:** Real-time net worth calculation.
* **Auto-Sync:** Persists data across sessions.
* **Privacy-First:** Zero cloud tracking. 

## 🛠️ Stack
* **UI:** Native CSS (Zero bloat)
* **Logic:** Vanilla JS (Pure performance)
